## Fontes para Seleção de Foto

Para identificar o padrão fotográfico editorial:
1. **ABC Color** — referência primária de curadoria visual
2. **Última Hora** — boa qualidade fotográfica
3. **La Nación** — cobertura ampla

Checar o que 2–3 desses veículos estão usando para aquela pauta específica, identificar o padrão e buscar a mesma foto na maior resolução disponível.
